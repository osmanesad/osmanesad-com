# osmanesad-com
osmanesad.com sayfasını ai kullanarak yeniden yazıyoruz.

### Framer to HTML-CSS
Siteyi framer platformundan alıp, sunucuya direkt html-css ve javascript kullarak yazdığımız bir basit, işlevsel profil sayfasına dönüştüreceğiiz.

Tüm bunları chatgpt'ye verdiğimiz komutlar, daha önceki sayfadan örnekler, cv, fotoğraflar, kodlar, medium blog yazılarınıda vererek bir karakter oluşturmasını isteyeceğiz.
